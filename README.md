# 🎓 Student Performance Prediction System

## 📘 Project Overview

A Machine Learning project that predicts whether a student will pass or fail based on study hours, attendance, and previous scores. The project demonstrates data preprocessing, model training, and GitHub-based version control.

---

## 🚀 Features

- Pass/Fail prediction
- Data preprocessing
- Exploratory data analysis
- Decision Tree model training
- Model evaluation
- GitHub project management

---

## 🛠️ Tools Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook
- Git & GitHub

---

## 📂 Files Included

- `Student_Performance_Prediction.ipynb`
- `Student_Performance_Dataset.csv`

---

## 🎯 Conclusion

This project showcases the application of Machine Learning in education by predicting student performance using academic and attendance-related factors.
